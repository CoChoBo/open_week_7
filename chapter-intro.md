## The command line
```bash

```

## Your first commit
```bash
git add .
git commit -m "first commit"

# "glass" 파일 에 아무 내용이나 추가한 뒤
git add .
git commit -m "second commit"

# 두 번째 커밋 상황에서는 아래도 가능:
git commit -am "second commit"
```

## Working together
```bash
git pull

# "students" 파일에 사용자 이름을 추가한 뒤
git commit -am "added student name"

git push
```
